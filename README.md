# Planejador Semanal de Tarefas

Projeto desenvolvido como desafio acadêmico da disciplina de React.

## Como rodar

```bash
npm install
npm run dev
```

## O que o projeto faz

Aplicação de organização de tarefas semanais no estilo Kanban. 
O usuário pode adicionar, concluir e deletar tarefas distribuídas por dia da semana.

## Tecnologias usadas

- React
- Vite
- CSS
